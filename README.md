# zetasql-npm-examples
This repo contains examples of usage of [zetasql-npm](https://www.npmjs.com/package/@fivetrandevelopers/zetasql) library

## Install dependencies

```sh
> npm i
```

## Compile project

```sh
> npx tsc
```

## Run project

```sh
> node ./out/index.js
```
